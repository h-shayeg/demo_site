Project

LatentMol

A generative model for de novo molecule generation, using a variational latent space conditioned on target physicochemical properties.

Machine Learning Chemistry Open Source MIT License
View on GitHub Read Documentation

Overview

Overview

LatentMol proposes novel candidate molecules by sampling and optimizing within a learned latent space, rather than searching directly over discrete molecular graphs. It was built as part of my dissertation work on generative approaches to molecular design, and is now maintained as a standalone library.

The Problem

Screening can't find what isn't there

Virtual screening can only evaluate molecules that already exist in some library. For genuinely novel chemical space — which is where a lot of the interesting solutions likely live — you need a model that can propose structures directly, not just rank existing ones. The hard part isn't generating something novel; it's generating something novel that a chemist could plausibly synthesize.

The Approach

Optimizing in a continuous latent space

LatentMol trains a variational autoencoder over molecular graphs, then performs gradient-based and Bayesian optimization within the resulting continuous latent space to search for candidates that satisfy target property constraints. A validity-aware decoding step and a synthesizability filter — built on a retrosynthesis-informed heuristic — discard chemically implausible proposals before they're ever shown to a user; generation without that filter produces a lot of technically-valid, practically-useless structures.

Generate Candidates Python 
from latentmol import LatentMol

model = LatentMol.from_pretrained("zinc250k")
candidates = model.generate(
    target={"logp": 2.5, "qed": 0.8},
    n_samples=50,
    filter_synthesizable=True,
)
