Project

ChemPipe

A reproducible, configuration-driven workflow framework for high-throughput computational chemistry screening.

Software Engineering Chemistry Open Source Apache 2.0 License
View on GitHub Read Documentation

Overview

Overview

ChemPipe turns a computational chemistry screening pipeline — descriptor generation, model inference, filtering, ranking — into a versioned, declarative workflow instead of a folder of one-off scripts. It grew out of a specific frustration: rerunning a two-year-old screening pipeline for a paper revision and discovering that half of it no longer ran.

The Problem

Pipelines that only run once

Most computational chemistry pipelines live as a loose collection of notebooks and shell scripts, glued together by memory of the order they're supposed to run in. Dependencies are pinned nowhere, intermediate outputs get overwritten, and the only complete record of what actually happened is the first author's laptop. That's a problem the moment a paper gets a revise-and-resubmit eighteen months later, or the moment anyone else on the team needs to extend the pipeline.

The Approach

Pipelines as versioned configuration

ChemPipe defines a pipeline as a YAML configuration of typed stages — each stage takes declared inputs, produces declared outputs, and is automatically re-run only when its inputs or code change, in the spirit of build tools like Make or Snakemake but purpose-built for cheminformatics data types (SDF, SMILES batches, descriptor tables). Every run is logged with its exact configuration and dependency versions, so a run from a year ago can be reproduced — or at minimum, diffed against — with a single command.

screening.yaml YAML 
stages:
  descriptors:
    script: gen_descriptors.py
    inputs: [raw_library.sdf]
    outputs: [descriptors.parquet]
  predict:
    script: predict_solubility.py
    inputs: [descriptors.parquet]
    outputs: [predictions.parquet]
  rank:
    script: rank_candidates.py
    inputs: [predictions.parquet]
    outputs: [ranked_candidates.csv]

# chempipe run screening.yaml --stage rank
