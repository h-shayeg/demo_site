# Personal Scientific Website

A personal scientific website built with Django to showcase research, projects, and educational content in chemistry, artificial intelligence, and software engineering.

## Tech Stack

* Django
* HTML
* CSS
* JavaScript

## Current Features

* Project initialized with Django
* Website application configured
* Home, About, and Contact pages
* Frontend template integrated
* Centralized template directory
* Static and media files configured

## Project Status

🚧 In Development

More features, including authentication, blog, portfolio, and research sections, are under active development.
