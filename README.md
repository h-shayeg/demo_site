# Personal Scientific Website

A personal scientific website built with Django to showcase research, software projects, scientific articles, and educational content in chemistry, artificial intelligence, and software engineering.

## Tech Stack

* Django
* HTML
* CSS
* JavaScript

## Current Features

* Django project initialized
* Modular application architecture
* Website application
* Blog application
* Project portfolio application
* Research application
* Home, About, and Contact pages
* Frontend template integrated
* Dynamic URL routing
* Centralized template directory
* Static and media files configured

```text
research/
```
